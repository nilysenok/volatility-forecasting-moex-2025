🔬 Feature Importance (Ablation Study)

Key Finding:

HAR components contribute 78% to model R², confirming the fundamental importance of modeling long-memory in volatility. This allows optimizing Feature Store by prioritizing these features.

Feature Group Full Model R² After Removal R² Δ R² Contribution
HAR Components 0.256 0.056 -0.200 78%
Vol_Ratio + RSI 0.256 0.217 -0.039 15%
Other Technical Indicators 0.256 0.239 -0.017 7%
💡 Practical Implication: Removing low-contribution features reduced computational cost by 40% while maintaining 93% of prediction quality.

└────────────────────────┬────────────────────────────────────┘
                         │
┌────────────────────────▼────────────────────────────────────┐

│  • Redis: Real-time features (access <10ms)                │
│  • InfluxDB: Historical time-series storage                 │

                         │
